# Дерево метаданных 1С: как читать и понимать

Книга‑учебник для абсолютных новичков, которые хотят научиться ориентироваться в любой конфигурации 1С **без программирования**.  
Мы учим читать дерево метаданных как чертёж деловой системы: видеть за справочниками, документами и регистрами реальные бизнес‑процессы.

## О чём эта книга

- **Не** учит писать код на встроенном языке 1С.
- **Не** справочник API или свойств объектов.
- **Да** — формирует навык «читать» чужую конфигурацию и проектировать свою.

Вы пройдёте путь от первого запуска Конфигуратора до осмысленного чтения незнакомой конфигурации из сотен объектов.  
Сквозной пример — торговая фирма «Канцтовары», для которой вы с нуля создадите работающее дерево метаданных.

## Для кого

- Абсолютный новичок без опыта программирования и без знакомства с 1С.
- Тот, кто уже видел 1С «изнутри» как пользователь, но хочет понять, как она устроена.
- Начинающий разработчик, который хочет получить прочный фундамент перед изучением кода.

## Как работать с материалами

1. Установите платформу 1С:Предприятие 8.3 (любая версия).
2. Склонируйте репозиторий или читайте файлы прямо на GitHub.
3. Открывайте соответствующий `.md`-файл модуля и выполняйте упражнения в Конфигураторе.
4. Идите от Модуля 0 к Модулю 9 — каждый следующий опирается на предыдущие.
